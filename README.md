# ArraysAndChart.js

Решения 10 заданий по использованию массивов и Chart.js

## Описание

Репозиторий содержит показательную практику и демонстрацию навыков работы с JavaScript, массивами и библиотекой Chart.js.

## Задания

Источник заданий от преподавателя:  
[Google Docs с заданиями](https://docs.google.com/document/d/16rpBMWLCoosbQ5cvChxsC9HSVx6dBuJrJqDYPnzzRx0/edit?tab=t.0)

## Технологии
- **JavaScript** 
- **Chart.js** - для построения графиков и диаграмм
- **Массивы** - различные методы работы с данными

## О структуре репозитория (проекта)

Все основные файлы находятся в корневом каталоге проекта:
- `index.html` - главная HTML-страница
- `mainStl.css` - основные стили
- `MainScr.js` - главный скрипт (базовые массивы, переменные и т.п.)

Решения заданий расположены в каталоге `Tasks/`.

### О проверке заданий

Для тестирования отдельных модулей доступно два варианта:

1. **Раскомментировать пути к ресурсам** в нижней части файла `index.html`
2. **Переключаться между коммитами** в истории Git для просмотра разных версий выполненных заданий


## Навыки

В рамках проекта отрабатываются:
- Работа с массивами (методы map, filter, reduce, etc.)
- Визуализация данных с помощью Chart.js
- Обработка и анализ данных
- Создание интерактивных графиков
- Адаптация дизайна под разные задачи
